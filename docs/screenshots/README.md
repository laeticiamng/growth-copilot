# Screenshots

Ce dossier contient les captures d'écran pour la documentation.

## Captures requises

1. **landing-hero.png** - Page d'accueil avec la section Hero
2. **cockpit.png** - Tableau de bord exécutif avec semaphores RAG
3. **hr.png** - Module RH avec annuaire des employés
4. **legal.png** - Module Juridique avec contrats

## Comment générer les captures

### Option 1 : Via navigateur

1. Ouvrir https://agent-growth-automator.lovable.app
2. Naviguer vers chaque page
3. Prendre une capture plein écran (1920x1080 recommandé)
4. Sauvegarder dans ce dossier

### Option 2 : Via les DevTools Chrome

1. Ouvrir les DevTools (F12)
2. Aller dans l'onglet "Device Toolbar" (Ctrl+Shift+M)
3. Définir la résolution à 1920x1080
4. Capture via "Capture full size screenshot"

## Dimensions recommandées

- **Desktop** : 1920x1080
- **Tablet** : 1024x768
- **Mobile** : 375x812

## Format

- Format : PNG
- Qualité : Haute
- Thème : Clair (Light mode)
